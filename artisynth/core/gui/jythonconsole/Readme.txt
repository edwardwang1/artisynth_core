To use the JythonConsole for Artisynth please follow these steps:

Download Jython version 2.1 from http://www.jython.org/download.html

After installation create a system environment variable JYTHON_HOME pointing
to the installation path.

eg. setenv JYTHON_HOME /usr/local/jython

Restart Artisynth, and the option to view the jython console should appear
under the view menu.

If the are any problems Please let Chad know.

@Known bugs: 

1. If you click on the close icon in to top right hand side of
the Jython Console it causes it to be out of sync with the View Menu option.
